# 🦙 Llama Peruana Interactiva

## ¿Qué es?
Una aplicación interactiva basada en una llama peruana
con inteligencia artificial.

## ¿Para qué sirve?
Permite conversar con una llama peruana virtual
que responde con personalidad peruana.

## 🛠️ Tecnologías
- C++
- Python
- Git / GitHub

## 📖 Guía de uso
1. Clona el repositorio
2. Compila el código
3. Ejecuta el programa
4. Habla con la llama!
